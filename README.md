## Project Overview
This project analyzes sales performance and profitability using Power BI to support data-driven business decisions.

## Tools Used
- Power BI Desktop
- Power Query
- DAX
- Excel / CSV data

## Key KPIs
- Total Sales
- Total Profit
- Profit Margin
- YoY Sales Growth

## Key Insights
- Top products contribute the majority of revenue
- Some regions show low profitability despite high sales
- Seasonal sales patterns are observed

## How to Use
- Clone this repository
- Open `Sales_Revenue_Analytics.pbix` in Power BI Desktop
- Update data source path if prompted
- Interact with slicers and visuals to explore insights
